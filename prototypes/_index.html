Milestone 3 Demo Slide — design variants

Each frame below is a live 16:9 mock of a possible redesign for the M3 "Enrichment" live-demo slide in ../slides.md. Round 3 (top row) adapts the recipe-card aesthetic from the lasagne reference — large hero image left, two stacked cream cards right that morph from Ingredients/Overview into Step-by-step/Done-looks-like as you advance.

Round 3 · Recipe-card variants (hero image + morphing stacked cards) · use ← / → / Space

V2a — Recipe Classic

Pattern · fixed hero · two stacked cards morph
Faithful adaptation of the lasagne layout. One hero image (cheapest production). Card A morphs Ingredients → Step-by-step; Card B morphs Overview → Done looks like.

V2b — Recipe Cinematic

Pattern · hero swaps per step · cards morph
Each step has its own crossfading hero image (5 assets total). Most visually arresting; highest production cost. Pairs well with recorded delivery.
Open full size ↗

V2c — Recipe Annotated

Pattern · ONE hero · pins light up per step
One hero image with four numbered pins positioned over distinct regions. Pin lights gold + pulses on the active step; previous pins check off. Production-friendly hybrid.

Round 2 · Interactive funnel variants · use ← / → or Space inside each frame

V1a — Stepper (focus + ambient)

Pattern · one big card · mise stays visible, dims
Active step occupies the right panel as a hero card. Mise-en-place panel highlights the prep relevant to the current step (workspace → skills → semantic → data).
Open full size ↗

V1b — Progressive Reveal (accordion)

Pattern · all 4 titles visible · only active expands
Students always see the arc of 4 steps as titles. The active step expands inline; finished ones collapse with a check; pending ones stay dimmed. Click any step to jump.
Open full size ↗

V1c — Spotlight (full-bleed)

Pattern · big step · checklist sidebar fills
Maximum focus mode. One step takes ~70% of the slide. Mise-en-place is a thin checklist that fills with checkmarks as you consume prep. Cleanest for in-room presentation.

Round 1 · Original static variants (for reference)

V1 — Mise en Place Card

Pattern · upfront tools + numbered steps
Medium-high text density · ordered · mirrors presenter flow exactly. Highest follow-along value for students typing in their own platform.
Open full size ↗

V2 — Parametric / TRN Matrix

Pattern · inputs × phases with focus-dim
High info-density · scannable · hover a row or column to isolate a track. Best for remote students who need the whole map at once.
Open full size ↗

V3 — Picture-Cook Flow

Pattern · iconographic flow, near-zero text
Low text density · highest visual recall · weakest for students who want to copy commands. Pairs with a confident verbal walkthrough.
