# Data Storytelling Workshop

**From Raw Data to Confident Narratives — Agent-Driven Data Storytelling**

UW Foster School of Business · Data Analytics · May 2026

---

## What This Is

A 1-hour workshop showing the difference between asking an AI to "analyze this data" and building a structured harness that produces defensible, auditable findings.

The workshop runs two VS Code workspaces side by side:

| Workspace | What's In It | Purpose |
|-----------|-------------|---------|
| `workspace-naive/` | Data files only, one-line CLAUDE.md | Baseline — what raw prompting gives you |
| `workspace-harnessed/` | Full skills suite, structured CLAUDE.md | What a disciplined harness produces |

The contrast between them is the whole lesson.

## The Scenario

Five UW campus coffee shops. Six months of transaction data. One vendor evaluation cycle. One social media audit. The data has intentional quality issues — inconsistent names, mixed date formats, missing vendor quotes, a suspicious anomaly at one shop, and a viral-post outlier in social data.

A second dataset (Seattle job market) is introduced in Milestone 4 to demonstrate context scoping.

## Prerequisites

To follow along during the workshop, you need:

- Any AI platform with file upload: Claude.ai, ChatGPT, Gemini, or GitHub Copilot
- A browser (to view HTML report outputs)

To run the harnessed workspace:

- Claude Code CLI or GitHub Copilot CLI
- Node.js 18+ (for Slidev, if running locally)

## Workshop Milestones

| Milestone | What Happens | Key Lesson |
|-----------|-------------|------------|
| [1. Naive Pass](milestones/01-naive-pass/README.md) | Load data, run one prompt | Raw AI analysis is useful but ungrounded |
| [2. Infographic](milestones/02-infographic/README.md) | Generate HTML visual report | Polished presentation is not the same as trustworthy analysis |
| [3. Enrichment](milestones/03-enrichment/README.md) | Skills + refinement interview | Structured frameworks surface what raw prompting misses |
| [4. Confident Narrative](milestones/04-confident-narrative/README.md) | Confidence scores, audience variants, spotlight tour | A harness produces analysis you can stake your name on |

## Custom Skills

Six workshop-specific agent skills are in `skills/`:

| Skill | What It Does |
|-------|-------------|
| `data-refinement-interview` | Pre-analysis interview — asks 4 targeted questions before drawing conclusions |
| `html-infographic-builder` | Generates a standalone HTML report with Chart.js and UW branding |
| `data-confidence-scorer` | Annotates every claim with confidence score, assumptions, and source lineage |
| `spotlight-walkthrough` | Injects a Shepherd.js guided tour into any HTML data report |
| `bootstrap-preflight` | Verifies the presenter or student environment is ready |
| `sync-skills-index` | Regenerates the AGENTS.md skills index after any skill changes |

Skills work with any instruction-following AI model. For platforms without native skill discovery, paste the `SKILL.md` contents into your system prompt.

## Data Generation

Data is synthetic and generated by `scripts/generate_data.py`. To regenerate:

```bash
python scripts/generate_data.py
```

Then sync to workspaces:

```bash
# macOS / Linux
cp data/coffee-shops/* workspace-naive/data/
cp data/coffee-shops/* workspace-harnessed/data/coffee-shops/
cp data/job-market/*   workspace-harnessed/data/job-market/

# Windows (PowerShell)
Copy-Item data\coffee-shops\* workspace-naive\data\ -Force
Copy-Item data\coffee-shops\* workspace-harnessed\data\coffee-shops\ -Force
Copy-Item data\job-market\*   workspace-harnessed\data\job-market\ -Force
```

## Running the Slides

```bash
npm install
npm run dev      # Dev server at http://localhost:3030
npm run export   # Export to PPTX (requires playwright-chromium)
```

Slides are also deployed to Vercel. See the repo description for the public URL.
